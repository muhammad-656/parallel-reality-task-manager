# Parallel Reality Task Manager

A React-based task management application with parallel reality features.

## Features

- Task management across multiple reality modes
- Analytics dashboard
- Calendar view
- Control panel for managing tasks

## Technologies

- React
- Tailwind CSS
- Modern JavaScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```
